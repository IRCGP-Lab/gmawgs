# GenoMycAnalyzer (GMA)

**GenoMycAnalyzer (GMA) CLI** is a comprehensive, automated bioinformatics pipeline designed for Whole Genome Sequencing (WGS) analysis of *Mycobacterium tuberculosis* (MTB).

It streamlines the entire process from raw data quality control to detailed clinical reporting, integrating multiple state-of-the-art tools and custom algorithms.

## ?? Quick Start

To install GMA immediately via Conda:

```bash
# 1. Create environment & Install package
conda create -n GMA -c shyun -c bioconda -c conda-forge gma

# 2. Activate environment
conda activate GMA

# 3. Check help message
gmawgs --help
```
